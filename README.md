## FagusArt

This is a presentational website for a furniture manifacturer.

Technologies used:

 - Client:  `Next JS(^13)` | `TypeScript` | `Tailwind-CSS` | `Material UI(^5)` | `SCSS` | `thirdweb` | `React Query` | `Node Mailer`
